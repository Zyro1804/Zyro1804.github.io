Objetivos del Sistema


General:

Sistema de muy bajo costo para pequeñas y medianas empresas, tanto la conexión concurrente con una pagina web para la publicidad del mismo y así ahorrar tiempos y costos en la empresa.

Un sistema finalizado para ser distribuido con todas aquellas personas que lo requieren, para así tener una competencia justa en el mercado.

Específicos:

• Aplicar encuestas en las paginas web dirigida a los clientes, para mejorar la publicidad y la forma de venta
• Tener un mantenimiento constante y mejoramiento del sistema
• Un sistema sencillo y entendible para cualquier persona
• De bajo costo para ser adquiridos por muchas empresas
• Conexión estable y fácil con paginas web de la empresa, o algún otro software de aplicación
